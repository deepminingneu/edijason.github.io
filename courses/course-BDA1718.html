Introduction to Big Data Algorithms

1st semester, 2017-2018

Overview

The Introduction to Big Data Algorithms includes brief introduction to big data, cloud computing techniques, distributed algorithms based on MapReduce and their applications, advanced MapReduce distributed algorithms, external memory based searching structures, tree and graph algorithms, big data mining, recommendation system and outsource algorithms, etc. Roughly the first 1/3 of the course focuses on cloud computing based on MapReduce; the second 1/3 focuses on query processing algorithms on large-scale data; the last 1/3 focuses on big data mining and recommendation problems.

This course seeks to provide students with new ideas about algorithmic design and analysis, and the frontiers of IT industry. It's also guided by the latest academic research results, and let the participating students know about the frontier knowledge of big data algorithms. Throughout the study and the task of this course, the students would master the basic ideas and technologies of algorithmic design and implementation in the cloud computing environment. Big data analysis is a hot topic in both academia and industry; while algorithm design is the core topic of big data. This course, based on the background of biomedical engineering, aims to educate high-end talents for intelligent medical, precise medical, personal health-care with big data analysis and mining skills.
